## Dev

```bash
bun install
```

To build:

```bash
bun run build
```

or watch changes:

```bash
bun run build:watch
```

To lint:

```bash
bun lint
```

To test:

```bash
bun test
```

To format:

```bash
bun fmt
```

To check by formatter, linter, import sorter

```bash
bun check
```

This project was created using `bun init` in bun v1.0.20. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
